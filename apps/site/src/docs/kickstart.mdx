import { toast } from "ezlet"

## Installation

```bash
bun add ezlet
# or
npm install ezlet
```

## Quick Setup

Add `Toaster` to your app's root layout, then call `toast` from anywhere. Base styles are included by the package import.

```tsx
import { Toaster, toast } from "ezlet";
export default function App() {
  return (
    <>
      <Toaster position="top-right" />
      <YourApp />
    </>
  );
}
```

## Action Toast

Toasts can include a button for user interaction.

<TabPanel
  preview={
    <PreviewBtn
      label="Try it"
      onClick={() => toast("File deleted", {
        action: { label: "Undo", onClick: () => toast.success("Restored") },
      })}
    />
  }
  code={`toast("File deleted", {
  action: {
    label: "Undo",
    onClick: () => restoreFile(),
  },
});`}
/>

## Promise Toast

Chain loading, success, and error states from a single promise.

<TabPanel
  preview={
    <PreviewBtn
      label="Try it"
      onClick={() =>
        void toast.promise(
          new Promise((resolve) => setTimeout(() => resolve("backup.sql"), 2000)),
          { messages: { loading: "Saving…", success: "Saved!", error: "Failed" } },
        )
      }
    />
  }
  code={`toast.promise(saveData(), {
  messages: {
    loading: "Saving…",
    success: "Saved!",
    error: "Failed",
  },
});`}
/>
